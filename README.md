# Disable Exclusive Control

This script is meant to disable the windows setting for allow apps to take exclusive control of audio devices.

The script has the following workflow.

1. Install Dependencies (WIP)
2. Pull GUID of current Default Audio Device and Default Communications Device (WIP)
3. Store the GUID into a variable (WIP)
4. Route the GUID of DAD and DCD to a variable (WIP)
5. Use the variable to check if the option is set correctly (WIP)
6. Apply it if it is not (WIP)
